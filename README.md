## üì¶ **Getting Started**

### 1. **Clone the repository**
```sh
git clone https://github.com/your-username/hunger-dash.git
cd hunger-dash
```

### 2. **Install dependencies**
```sh
npm install
```

### 3. **Run the development server**
```sh
npm start
```
Visit [http://localhost:1234](http://localhost:1234) to view the app.

### 4. **Run tests**
```sh
npm test
```
